# 解説

## レジスタ

* rbp :ベースポインタ（フレームポインタ）
* rsp : スタックポインタ（スタックのトップアドレスを格納）
* rsi : 
* rdi : 
* eax : 
* edx : 
* rip : 
## main
### `pushq %rbp`

動作<br>
* rsp = rsp-8
* *(rsp) = rbp

rbp(呼び出し元のベースポインタ）の値をスタックトップに退避<br>
この時点で、スタックは以下のイメージ
```
+------------------+
| 呼び出し元のrbp    |   ← rsp
+------------------+
| return address   |   ← 呼び出し元の次の命令アドレス
+------------------+
```

### `movq  %rsp, %rbp`

rspをrbpに格納

### `call addNum`

動作<br>
①戻りアドレス(mainの次の命令アドレス)をスタックにpush<br>
* rsp = rsp-8
* \*(rsp) = mainの次の命令アドレス

②プログラムカウンタの更新(呼び出し関数へのジャンプ)<br>
* rip = addNum
