# ディレクトリ移動
* ホームディレクトリに移動
```bat
cd %homepath%
```
* ドライブ間移動
```bat
rem CドライブにいるときにDドライブに移動
D:
```
***

# ディレクトリ表示
```bat
dir /B
```
`/B`:ファイル名のみ表示する
***

# ファイル操作
## xcopy
* ディレクトリをコピーする
```bat
xcopy /H /S <SRC_DIRECTORY> <DST_DIRECTORY>
```
`/H`: 隠しファイルを含む
`/S`: 空でない場合、サブディレクトリも含む
linuxの`cp -r`に相当
***

# 変数設定
## set
環境変数を設定する
```bat
set PATH=C:\Users\username\Scripts;$PATH
```
***

# シンボリックリンク
* ディレクトリのシンボリックリンクを作成
```bat
> mklink /D <新規リンク> <ターゲットパス>
```
`/D`: ディレクトリのシンボリックリンクを作成
注) リンク、ターゲットの指定順は`ln -s`とは逆なので注意

***

# コマンドのマクロ(エイリアス)を設定(doskey)
`doskey`を使う
* マクロの設定
```bat
> doskey dir = dir /B
```

* 現在のセッション(プロセス)で設定されているマクロ一覧を出力する
```bat
> doskey /macros
rem dir=dir /B
```

## net use
ネットワーク上の共有フォルダをローカルドライブとして割り当てしたり、アクセスしたりするためのコマンド。
これを使うと、ネットワーク上の別のコンピュータやサーバーにあるフォルダを、ローカルドライブのように使えるようになる。
https://zenn.dev/platinadata/articles/ca39a59ef76492<br>
<br>
* Xドライブに\\192.168.1.0\sharedを割り当てる
```bat
net use X: \\192.168.1.0\shared
```
* ネットワークドライブや接続されている共有フォルダの一覧を表示する
```bat
net use X: /delete
```

* ネットワークドライブや接続されている共有フォルダの一覧を表示する
```bat
net use
```
