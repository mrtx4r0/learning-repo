# DHCP
* Dynamic Host Configuration Protocol
* IPアドレスを自動的に割り当てるプロトコル
* DHCPサーバ/DHCPクライアントからなる
  * DHCPサーバ: Windowsサーバ、Linuxサーバ、Ciscoルータ、家庭用ルータなど
  * DHCPクライアント: WindowsなどのPC、IP電話など
* DHCPパケット
  * UDP上で動作する
  * DHCPサーバ宛てパケットの宛先ポート番号は「67」（UDPヘッダにポート番号のフィールドがある）
  * DHCPクライアント宛てパケットの宛先ポート番号は「68」（同上）
  * DHCPメッセージ部分に、割り当てられるIPアドレス情報が書かれる
* DHCP通信の仕組み
  * DHCPクライアント → ブロードキャストアドレスにDHCP Discoverメッセージを送信（最初は自身のIPアドレス、DHCPサーバのIPアドレスいずれも知らないため）
