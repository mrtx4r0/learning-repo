# IP
## IPヘッダ
主なフィールド<br>
(下位ビットから)<br>
* フラグ(flags) : 3bit。パケット分割の制御情報。パケットの分割を許可するか/分割された最後or途中のフラグメントか
* 生存期間(TTL) : 8bit。パケットの生存期間を示す。1台のルータを通過するごとにTTL値は1ずつデクリメントされ、0になるとパケットが破棄される
* プロトコル(protocol) : 8bit。上位層(トランスポート層)のプロトコルが何であるかを示す。ICMP, OSPFなど同じ階層のプロトコルを示す場合もある。
* 送信元IPアドレス(source address) : 32bit
* 宛先IPアドレス(destination address) : 32bit

# ルーティング
## ルータ
* ルーティングテーブルを管理し、宛先IPアドレスを基に適切なネットワークにパケットを転送する
* NAT/NAPT機能、DHCP機能、アクセスポイント機能などもある
* AS:ルーティングポリシを決めてそれを適用して運用する範囲
* IGP:AS内で使うルーティングプロトコルの総称
* EGP:AS間で使うルーティングプロトコルの総称
