## Exコマンド
`:`で始まるコマンド(=コマンドラインモードで使用するコマンド)

### `:s`(:substitute)
* 書式
```vim
:[行範囲]s/{置換前文字列}/{置換後文字列}/フラグ
```
* 例
```vim
:%s/foo/bar/g
```
`%`: 全行. `1,$`も同じ<br>
`g`: sコマンドのオプション. 各行で一致するすべてのパターンに適用する<br>


### `:g`(:global)
*  書式
```vim
g/{pattern}/{command}
```
* 意味
patternで始まる行それぞれに対して、commandを実行する

* 例
```vim
g/^#/p  " #で始まる行を表示する
```

### `:exe`(:execute)
* 書式
```vim
execute {string}
```
* 意味
文字列を組み立てて、それをコマンドとして実行する

* 例
```vim
g/^#/exe "normal 2ko***\<Esc>"
```
#で始まる行に対して、normalモードで2ko\*\*\*(2行上に行を挿入し「\*\*\*」を入力。その後<Esc>でnormalモードに戻る<br>
※「\<ESC>」とバックスラッシュを入れているのは、文字列と解釈させないため<br>


## インデント操作
shiftwidth(sw)オプションの設定値分インデントをずらすことができる<br>
`:set shiftwidth?` : 設定されている値がわかる（デフォルトは8)<br>
`<<` : 左にずらす<br>
`>>` : 右にずらす<br>
`:set tabstop=4` : tab幅の変更<br>

## 複数行への文字一括挿入
|操作|コマンド|備考|
|---|---|---|
|3〜10行目の行末に"\<br\>"を挿入|:3,10 normal A\<br\>|normal:通常モードのキー操作を複数行に適用するコマンド<br>A:行末に追加|

## 単語単位で削除
`diw`:&ensp;カーソルが乗っている単語を削除

## 行内の文字検索(カーソル移動)
`f{char}`:&ensp;行内の右方向に{char}を検索し、存在すればカーソル移動する<br>
`F{char}`:&ensp;行内の左方向（以下同上）<br>

## レジスタを使った操作
#### ヤンクレジスタ
* 記号`0`で指定する
* `y{motion}`コマンドを実行すると、指定されたテキストは無名レジスタと「ヤンクレジスタ」にコピーされる
* ヤンクレジスタにコピーされたテキストは`"0P`(`"0p`)で貼り付け可能

## netrw(ファイラ)の操作
`<C-w> h/j/k/l` : ウィンドウを移動<br>


## Tips・コマンド集
「# 」で始まるすべての行の2行上に行を挿入し、その行に「\*\*\*」を入力する
```vim
:g/^#/exe "normal 2ko***\<Esc>"
```
