# 1.02.1_ファイルの所有者とパーミッション

#### スティッキービット
スティッキービットが設定されていると、そのディレクトリに対して書き込み権限を持つユーザでも、自分が所有していないファイルの削除を禁止される。
スティッキービットは、以下の`/tmp`の例のようにパーミッションのその他のユーザの実行権限の位置に`t`が表示される
(linuxでは通常、`/tmp`ディレクトリにスティッキービットが設定されている)

```bash
$ ls -dl /tmp
drwxrwxrwt 11 root root 220 Mar 14 07:59 /tmp
         ~
```
