## 変数展開
#### `${FOO#PATTERN}`
変数`FOO`に格納された文字列の**先頭部分**と`PATTERN`がマッチしたら、最も**短く**一致する部分を取り除いた残りの部分を返す<br>
#### `${FOO##PATTERN}`
変数`FOO`に格納された文字列の先頭部分と`PATTERN`がマッチしたら、最も**長く**一致する部分を取り除いた残りの部分を返す<br>
使用例<br>
```bash
$ SRCPATH=/Users/user/C/src/helloworld.c
$ echo ${SRCPATH##/*/} # 出力結果: /Users/user/C/src/
```
#### `${FOO%PATTERN}`
変数`FOO`に格納された文字列の**最後の部分**と`PATTERN`がマッチしたら、最も**短く**一致する部分を取り除いた残りの部分を返す<br>
使用例<br>
```bash
$ SRCPATH=/Users/user/C/src/helloworld.c
$ echo ${SRCPATH%/*} # 出力結果: helloworld.c
```
