## ネットワークコマンド
`ip link show` : ネットワークインターフェースの物理的な状態や基本設定を表示（ip addr showより低レベル）<br>
`ip addr show `: ネットワークインターフェースに割り当てられているIPアドレスやサブネット情報を表示する<br>
`ip route show ` : ルーティングテーブルを表示<br>
`sudo ip netns add ns1` ns1という名前のNetworkNamespace<br>
`sudo ip netns list` : network namespaceの一覧<br>
`sudo ip netns exec ns1 ip addr show` : ns1というnetwork namespaceの情報を表示<br>
`sudo ip netns delete ns1`　ns1 というnetworknamespaceを削除<br>
`sudo ip link add ns1-veth0 type peer name ns2-veth0` : ペアとなるveth(virtual ethernet device)の作成<br>
`sudo ip link set ns1-veth0 netns ns1 `: 作成したvethをnetwork namespace に所属させる<br>
`sudo ip netns exec ns1 ip addr add 192.0.2.1/24 dev ns1-veth0`: ns1-veth0にnetworknamespace上でIPアドレスを付与する<br>
`sudo ip netns exec ns1 ip link set ns1-veth0 up` : ip link setはネットワークIFのstate(管理上の)のUP/DOWNを設定する<br>
